# AI Rule-Based Chatbot

## Overview

AI Rule-Based Chatbot is a simple chatbot application that responds to user queries using predefined rules and keyword matching techniques. The chatbot provides instant responses to common questions related to Artificial Intelligence, Machine Learning, programming languages, and general conversations.

## Features

* Interactive chatbot interface
* Rule-based response generation
* AI and Machine Learning related responses
* Date and time queries
* Typing effect simulation
* Clear chat functionality
* Chatbot avatar and modern UI
* Auto-scroll chat window
* Enter key support

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

AI-Rule-Based-Chatbot/

## How It Works

1. User enters a message.
2. The chatbot checks the message against predefined keywords.
3. A matching response is generated.
4. If no keyword is found, a default response is displayed.
